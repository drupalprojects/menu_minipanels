
Allows an administrator to specify a minipanel to be associated with a Drupal menu item.
When that menu item is hovered or clicked (as per config), the minipanel content will be shown
using the qTip javascript library

Installation
------------

1. Add the module to your modules directory
2. Visit http://craigsworks.com/projects/qtip/download/ and download the *minified* qTip version. 
   The name of the file you need from the archive is jquery.qtip-1.0.0-rc3.min.js.
3. Place this file in the js/lib subfolder
4. Enable the module
5. Visit "Menu Minipanels Settings" to confgure default settings for qTip
